# Biblioteca Virtual Django

Sistema web para gerenciar livros, empréstimos e usuários de uma biblioteca, desenvolvido com Django e Bootstrap.

## Funcionalidades
- Página inicial com links rápidos para cadastro e visualização de livros.
- Cadastro de livros.
- Listagem de livros.
- Realização de empréstimos.
- Visualização de empréstimos realizados.
- Pesquisa de livros.

## Tecnologias utilizadas
- Python 3.x
- Django 4.x
- Bootstrap 4/5 (front-end)
- django-widget-tweaks (para customização de formulários)
